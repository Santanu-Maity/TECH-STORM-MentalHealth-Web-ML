# TECH STORM - MentalHealth-Web-ML

## Overview
This project is an AI-powered web application developed by **Team TECH STORM** for mental health counselling. 
It uses user profile and mood data to calculate a Mental Health Score and provides personalized recommendations. 
If needed, the system can book appointments with doctors and send emergency alerts.

## Features
- User signup/login & profile completion
- Mood tracking and mental health score calculation
- Personalized coping strategies and lifestyle recommendations
- Motivational quotes, yoga & meditation guides
- Music and movie suggestions for mental wellness
- Doctor appointment booking (anonymous or full profile)
- Emergency alerts for high-risk users
- Dashboard with charts & progress tracking

## Tech Stack
- Frontend: HTML, CSS, JavaScript, Bootstrap / React.js
- Backend: Python (Django / Flask) or Node.js + Express
- Database: Firebase / MySQL / PostgreSQL
- Machine Learning: Python, scikit-learn, TensorFlow/PyTorch
- Deployment: Netlify / Firebase Hosting / Heroku
